# 学习路线

你已经熟练

- JavaSE：OOP
- JavaWeb：独立开发的MVC 三层架构网站
- SSM（SpringMVC + Mysql)

本章将讲解

- Springboot
- Springcloud

什么是Spring? S pring是为了解决企业级应用开发的复杂性而创建的，简化开发的一个开源框架。

为了降低Java开发的复杂性，Spring采用了以下4种关键策略：

1. 基于POJO的轻量级和最小侵入性编程；
2. 通过1OC，依赖注入(D)和面向接口实现松耦合；
3. 基于切面(AOP)和惯例进行声明式编程；
4. 通过切面和模版减少样式代码；

不要忘记，Spring 中 约定大于配置。
