                        # Cost Function

                        神经网络的训练目标是最小化 总误差函数「代价函数」 E(w)E(\boldsymbol{w})E(w)，其中 w\boldsymbol{w}w 表示神经网络的所有权重和偏置。

                        总误差函数可以根据具体的问题而不同，常见的包括均方误差、交叉熵等。

                        为了简化任务，假设 神经网络 N 以使用 均方误差+线性神经元，且

                        • 有 K 个神经元输出，一个样本向量输入
                        • 输出层中，第 k 个神经元的预期结果为 yky_{k}yk​
                        • 输出层中，第 k 个神经元的实际输出结果为 hΘ(x)kh_{\Theta}(x)_{k}hΘ​(x)k​

                        J(θ)=12∑k=1K(hΘ(x)k−yk)2 J(\theta)=\frac{1}{2} \sum_{k=1}^{K}\left(h_{\Theta}(x)_{k}-y_{k}\right)^{2} J(θ)=21​k=1∑K​(hΘ​(x)k​−yk​)2

                        其中

                        • J(θ)J(\theta)J(θ) 是 关于 hΘ(x)kh_{\Theta}(x)_{k}hΘ​(x)k​ 的函数
                        • 在 xk 和 yk 已知的情况下，hΘ(x)kh_{\Theta}(x)_{k}hΘ​(x)k​ 是关于 θi\theta_iθi​ 的函数

                        因此 J(θ)J(\theta)J(θ) 是关于 θi\theta_iθi​ 的函数，应当不断调整每个 θi\theta_iθi​ 的值，使用梯度下降找到 J(θ)J(\theta)J(θ) 的最小值。这需要对每个 θi\theta_iθi​ 求偏导数。

                          • 角标 j 的层关系，指的是当前层和下一层 j + 1
